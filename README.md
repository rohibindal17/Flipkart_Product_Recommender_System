# 🚀 LLMOps Deployment with Docker, Kubernetes, Minikube & Monitoring

A complete end-to-end **LLMOps pipeline** to containerize, deploy, and monitor an AI/ML application using **Docker, Kubernetes (Minikube), Prometheus, and Grafana** on a Google Cloud VM.

---

## 🔥 One-Line Description
Deploy and monitor your ML app on Kubernetes using a fully production-style LLMOps pipeline.

---

## 📌 Overview

This project demonstrates how to:
- Containerize an ML app using Docker  
- Deploy it on Kubernetes using Minikube  
- Manage secrets securely  
- Expose services for external access  
- Monitor application performance using Prometheus & Grafana  

---

## 🛠️ Tech Stack

- **Python (Flask)** – Backend application  
- **Docker** – Containerization  
- **Kubernetes (Minikube)** – Container orchestration  
- **Google Cloud Platform (GCP VM)** – Infrastructure  
- **Prometheus** – Monitoring & metrics collection  
- **Grafana** – Visualization dashboards  
- **GitHub** – Version control
